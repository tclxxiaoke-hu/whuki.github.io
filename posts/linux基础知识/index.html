Linux

入门篇

  • / 根目录
  • boot 引导分区 系统启动使用一般
  • swap 交换分区 单独的一块分区作为内存使用
  • xfs 高性能日志文件系统

文件目录

  • /bin =》/usr/bin 二进制目录 直接可以执行的常见命令 ,是指向 /usr目录下
  • /sbin 管理员执行的二进制
  • /lib /lib64 库目录 这些是系统使用的库
  • /usr 用户所需要级别的东西都放到这个目录
  • /boot 引导分区目录
  • /dev 设备目录 cpu、硬盘、鼠标等硬件设备文件
  • /etc 系统管理的配置文件
  • /home 普通用户的文件夹
  • /root 超级管理员的主目录
  • /opt 可选目录 第三方软件会安装到这个目录
  • /media u盘、光驱等可移动设备
  • /mnt 挂载目录
  • /proc 进程目录 系统相关
  • /run 临时目录 当前记录系统信息
  • /svr 系统服务相关
  • /sys 系统硬件相关信息
  • /tmp 临时目录
  • /var 可变目录 存放各种日志

vim

  • 一般模式 删除、复制、粘贴s

  • 编辑模式 输入i、a、o进行文本的编辑

  • 命令模式 :wq :q :q!

    • u 撤退操作
    • ​
    • shift+^ 快速移动到行头部
    • shift+$ 快速移动到行尾
    • gg 快速移动到文件头部
    • G 快速移动到文件尾部
    • ​
    • [n] yy 复制该行
    • y$ 复制当前光标到结尾的内容
    • y^ 复制当前光标之前的内容
    • yw 复制该单词
    • ​
    • shift+X 删除之前的内容
    • p 复制到当前位置
    • [n] dd 删除该行
    • ​
    • I 在该行的最前面插入
    • A 在该行的最后插入
    • o 在当前行的下一行插入
    • O 在当前行的上一行插入
    • ​
    • :set nu 显示行号
    • /[内容] 查找值
    • :s/[内容]/[替换的内容] 仅替换当前行的内容
    • :%s/[内容]/[替换的内容]/g 全文替换

文件操作命令

  • less 查看大文件内容 http://c.biancheng.net/view/4004.html
  • head -n 【行数】 文件名称 查看文件前n行的内容
  • tail -n 【行数】 文件名称 查看倒数n行的内容
  • tail -f 文件名称 查看当前文件最新追加的内容

用户管理命令

  • useradd 用户名 添加新用户
  • useradd

磁盘

  • du -sh 查看当前目录的大小

  • df -h 查看磁盘剩余空间

  • free -h 查看内存占用情况

  • lsblk -f 查看设备上挂载的大小

  • fdisk 查看磁盘分区的情况

    • fdisk -n 进行新磁盘的分区

  • mkfs 磁盘分区格式化

  • mount 【设备名称】 【目录名称】 将磁盘设备挂载到目录上

系统进程操作

  • ps aux | grep xxx

  • ps -ef | grep xxx

  • top 实时的监控信息

    • -p 查看进程
    • M 查看内存

  • netstat -anp | grep 进程号 查看该进程网络信息

  • netstat -nlp | grep 端口号 查看网络端口号占用情况

shell 编程

  • 定义变量 变量名称=值 两个不能为空

    • echo $变量名称 打印变量
    • export 变量名称 设置变量为全局变量
    • readonly 变量名称=值 只读变量
    • unset 变量名称 撤销变量
    • $【数值】 终端输入的变量
    • $# 获取输入变量的个数
    • $* 获取命令行中所有的参数，并把参数看作一个整体
    • $@ 命令行中所有的参数
    • $? 查看脚本是否执行成功 0 表示成功 其它数值表示出错

  • 运算符

    • expr 运算
    • $[运算式]

  • 条件判断

    • test 【判断式】 0 表示成功 1 表示失败
    • [ 判断式 ] 判断式前后一定要有空格
    • 比较 lt gt ge …
    • if [条件] ; then
    • 程序
    • fi
